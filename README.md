# Autonomous Software Agents

This repository is being created for the project of the Autonomous Software Agents course at University of Trento. 

The core goal of this project, is to create and develop an agent that is able to resolve a series of send-delivery packages, taking as input the starting position and nothing more.

I.e. the agent starts the task totally blind, meaning that it has to discover and resolve by himself the positions of each packages and delivery point.
